UWaterloo Food Services Daily Menu

The UWaterloo Food Services Menu that I developed is a Racket program that returns the lunch and dinner options that are being offered by the residence cafeterias at UWaterloo. The program takes user input to decide which restaurant's menu you want to look at, then uses the UWaterloo API to retrieve that information. The information is then parsed for the current day's menu which is then formatted and displayed. Click Here to visit this repository on Github.
